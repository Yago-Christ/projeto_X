# PROJETO X 1.0 - Convite de Festa

Página React para convite de festa de 18 anos com design moderno e responsivo.

## Tecnologias Utilizadas

- React 18 com hooks (useState)
- Tailwind CSS para estilização
- Design mobile-first
- Efeito de vidro fosco (glassmorphism)

## Como Executar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm start
```

3. Abra [http://localhost:3000](http://localhost:3000) para visualizar.

## Funcionalidades

- Campo de input para nome do convidado
- Dois botões de ação (pagar agora / reservar)
- Mensagens de confirmação dinâmicas
- Design responsivo para mobile e desktop
- Animações e transições suaves
- Paleta monocromática elegante
